Predicting Stress Levels Using Wearable Health Metrics

Project Overview

This project explores whether metrics of wearable health technology can be used to predict users stress levels. Biometric data such as sleep quality, heart rate variability (HRV), activity levels, and other health indicators will be utilized to investigate which factors are most strongly associated with stress.

The project was completed as part of a graduate capstone in Data Science and focuses on applying statistical and machine learning methods to real-world health data.
